# LAMMPS

- [Homepage](http://lammps.sandia.gov/index.html)

## Setup(win 64bit)

1. Download package
1. Extract to any folder
1. Add bin folder to system path
1. run sample with `lmp_serial -in in.test`

## Run

### multicore (need win pw set)

`mpiexec -np 4 lmp_mpi -in in.chain`
